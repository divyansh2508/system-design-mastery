# System Design — Knowledge Base (Markdown)

The **plain-text mirror** of the interactive site. Each file is the full lesson — prose, tables, worked examples, and video links — with all the HTML/CSS/JS stripped out. Readable anywhere, greppable, and easy to drop into Obsidian/Notion or feed to an LLM.

> The **live version** (with the interactive simulators) lives in the parent folder — open [`../index.html`](../index.html). These `.md` files are the knowledge; the simulators are the intuition.

~110,000 words across 28 modules.
