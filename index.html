How to use this

  1. Host it locally. Easiest: double-click index.html to open it in your browser — every link works. For a smoother experience (and saved progress across pages), run a tiny server: cd "System Design" && python3 -m http.server, then open localhost:8000.
  2. Work the roadmap top to bottom. Each module runs the same loop: read the notes → play the tool → drill the case study → review. Check modules off on the roadmap as you go.
  3. Everything's yours. These are plain HTML files in Documents/Job Search/System Design/ — keep them, edit them, back them up. New tools and modules drop straight into this folder and appear here.
  4. Stuck? Ask. Any concept that won't click, any tool you want changed — just ask and we'll fix it or build a new one.
