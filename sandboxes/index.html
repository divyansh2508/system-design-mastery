Docker Sandbox Labs

The simulators build your intuition; these build your hands. Each lab spins up real infrastructure — actual Nginx, Postgres, Redis, Kafka, Prometheus, Keycloak — in Docker, then walks you through breaking and observing it. This is what turns interview-real into job-real.

⚠ Prerequisites

You need Docker Desktop installed and running. Each lab is a folder under System Design/sandboxes/ — open its guide, then cd into that folder and follow the commands. Tear down with docker compose down -v when done.
